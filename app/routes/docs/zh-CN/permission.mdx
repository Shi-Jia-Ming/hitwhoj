# 权限控制

本站把权限写死了喵

## 用户 (User)

用户有四种不同的角色：

- 超级管理员 (Su)
- 管理员 (Admin)
- 普通用户 (User)
- 封禁用户 (Banned)

其中封禁用户会被禁止登录。

## 题目 (Problem)

题目有两种不同的权限控制方式，当你创建题目的时候，你可以选择其中一种，但是不能改变现有题目的权限控制方式。

| 题目类型 | 设置为私有                | 设置为公开                  |
| :------: | ------------------------- | --------------------------- |
| 个人题目 | 仅题目创建者可以查看/提交 | 所有人均可以查看/提交       |
| 团队题目 | 仅团队管理员可以查看/提交 | 团队内所有人均可以查看/提交 |
